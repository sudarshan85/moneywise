# MoneyWise Mobile Responsive + PWA Implementation Plan

## Overview
Make MoneyWise work on mobile devices and installable as a homescreen app.

## Requirements
- Dashboard and Transactions are primary pages
- All pages must be accessible and usable on mobile
- No offline support needed
- Add to Homescreen capability (PWA)

---

## PHASE 1: PWA Setup

### 1.1 Create manifest.json
Location: `frontend/public/manifest.json`

```json
{
  "name": "MoneyWise",
  "short_name": "MoneyWise",
  "description": "Personal envelope budgeting app",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#1a1a2e",
  "theme_color": "#F5A623",
  "icons": [
    {
      "src": "/icons/moneywise_icon.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "/icons/moneywise_icon.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
```

### 1.2 Update index.html
Add to `<head>`:
```html
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#F5A623">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
<link rel="apple-touch-icon" href="/icons/moneywise_icon.png">
```

@media (max-width: 768px) {

    grid-template-columns: repeat(2, 1fr);

## PHASE 4: Transactions Mobile CSS

### 4.1 Update Transactions.css
Key changes:
- Stack filter controls vertically
- Hide less important columns (Account, Actions)
- Make table horizontally scrollable

## Design Decisions

1. **Bottom Navigation**: Using fixed bottom nav on mobile (more thumb-friendly than top tabs)
2. **Horizontal Scrolling**: Tables scroll horizontally rather than hiding all columns
3. **Stacked Forms**: Filter controls and forms stack vertically on mobile
4. **No Category Details Panel**: Hide on mobile to save space (users tap category rows)
